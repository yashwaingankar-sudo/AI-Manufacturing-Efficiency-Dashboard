## 📌 Internship Project
This project was developed during my internship, focusing on analyzing manufacturing efficiency using data visualization and analytical techniques.

---

## 🧩 Problem Statement
Manufacturing industries often face inefficiencies due to machine downtime, defects, and network delays. This project aims to identify these issues and improve operational efficiency using data-driven insights.

---

## 📊 Overview
This Power BI dashboard analyzes manufacturing efficiency using sensor, production, and network data. It helps uncover key drivers such as machine performance, defect rates, and latency affecting overall productivity.

---

## 🚀 Features
- Interactive dashboard for monitoring machine performance  
- Visual analysis of defect rates across production units  
- Identification of latency issues impacting efficiency  
- Drill-down insights for better operational decision-making
