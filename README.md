## 📌 Internship Project
This project was developed during my internship and focuses on integrating **AI-driven analytics with Power BI** to monitor and improve manufacturing efficiency using sensor, production, and network data.

---

## 🧩 Problem Statement
Manufacturing teams face multiple challenges:
- Delayed detection of efficiency degradation  
- Manual interpretation of large volumes of sensor and network data  
- Lack of automated and interpretable efficiency assessment  

### ⚠️ Without AI Support:
- Efficiency drops go unnoticed  
- Corrective actions are delayed  
- Production losses increase  

---

## 🎯 Objective
To build a system that:
- Automatically classifies manufacturing efficiency (High / Medium / Low)  
- Identifies key drivers affecting performance  
- Enables faster, data-driven decision-making  

---

## ⚙️ Methodology

### 🔹 Data Preprocessing
- Time-based ordering (Date + Time)  
- Encoding categorical variables (Operation Mode)  
- Feature scaling for numerical data  
- Handling class imbalance  

### 🔹 Feature Engineering
- Sensor stability indicators  
- Energy efficiency ratios  
- Error-to-output ratios  
- Network reliability score  

### 🔹 Model Development
- Baseline: Logistic Regression (multi-class)  
- Advanced Models:
  - Random Forest Classifier  
  - Gradient Boosting / XGBoost  

### 🔹 Model Evaluation
Models were compared based on:
- Accuracy  
- Stability  
- Interpretability
